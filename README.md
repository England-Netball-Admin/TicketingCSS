# England Netball Ticketing Site header and footer (HTML + CSS) 


📌 Purpose

This repository contains the reusable header and footer components of the EN - the elements are designed to be easily integrated into other EN projects to ensure a consistent look and feel across pages.
These components are intended for internal use only.


⚙️ Features

✅ Single HTML file containing both header and footer
✅ Modular CSS (3 files)
🚫 No JavaScript
🚫 Not for external use


📁 File Structure

en_tickets_hf_html
    header_footer.html
    style-0-reset.css
    style-1-brand.css
    style-2-header-footer.css


🛠️ Usage

Include the header and footer on pages by copying the HTML file 
and link the relevant CSS file/s in the '<head>' of your page/s
